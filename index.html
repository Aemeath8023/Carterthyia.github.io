// Lore & Background

故事背景

序章

星光降临

在一个被黑暗侵蚀的世界，一颗异星的光芒穿越虚空坠落。那一夜，所有守望者都看见了那道划破天际的星轨——卡提希娅，就此诞生于光与影的交界。

第一章

双生契约

卡提希娅在流浪途中邂逅了神秘的数字存在爱弥斯（Aemeath）。两者之间形成了跨越物质与数字界限的羁绊，这份契约赋予了她双形态切换的能力。

第二章

温柔的代价

每一次切换形态都会消耗卡提希娅的星光能量。为了守护挚友，她在与强敌的战斗中几度陷入能量枯竭的边缘。然而正是这份温柔，成为了她最强大的武器。

现在

守光者

如今，卡提希娅与爱弥斯共同守护着这片星光。她的故事还在继续书写，每一个造访这里的人，都将成为她故事中的一部分。

// Game Guide

游戏攻略

⚡

形态切换时机

建议在敌方蓄力时切换战斗形态，打断节奏并触发反击连锁。切换 CD 约 8 秒，合理规划节奏。

✨

星光能量管理

保持能量槽在 40% 以上方可安全切换。低于此值时优先使用治愈技能回充，避免形态锁定。

🛡

护盾优先级

星光护盾可叠加两层，建议在 Boss 战前提前叠满。护盾存续 12 秒，优先保护脆皮输出位。

🎯

仇恨控制

卡提希娅治愈量大，容易吸引仇恨。配合沉默技能或位移技能，可有效规避集火风险。

🔥

速攻连招

星光冲击 → 形态切换 → 终极技 → 普攻 ×3。此连招在 3 秒内可打出约 280% 攻击力的爆发伤害。

💫

持续输出流

维持治愈形态，利用被动星光再生持续回血，配合队友控制技能稳步推进，适合长战。

🌙

双形态循环

战斗形态 2 技能 → 切换治愈 → 大治愈 → 切换战斗 → 终极。约 20 秒一次完整循环。

💍

主流输出配装

星辉之核（主武器）+ 光辉圣冠 + 星河碎片 ×2。侧重攻击词条，暴击率维持在 75% 以上。

🌿

辅助治愈配装

温柔之心（主武器）+ 生命织网 + 星光护符 ×2。生命值词条优先，治愈加成可达 +45%。

⚖️

均衡配装

晨星之刃 + 双属性混搭套装。适合新手前期过渡，兼顾输出与续航，上手门槛低。

👥

标准阵容

卡提希娅（治愈核心）+ 高爆发 DPS ×2 + 控制辅助。卡提希娅专注回血与护盾，DPS 负责爆发。

💥

双核爆发阵

卡提希娅（战斗形态主C）+ 爱弥斯联动 + 辅助×2。触发双生契约被动，爆发伤害提升 35%。

🏰

龟速稳进阵

卡提希娅 + 重甲坦克 + 群控 + 远程 DPS。适合高难度副本，牺牲速度换取极高容错率。
